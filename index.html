Impariamo a caricare un progetto su GitHub

Questi sono gli step da seguire nell'ordine seguente:

  1. creare e dare un nome (senza spazi) al repository su GitHub

  2. Creare il documento e salvarlo Control + S

  3. Andare su Source Control e nel file premere il simbolo + (Stage Changes)

  4. Adesso bisogna inserire il messaggio che si vuole dare alla Commit dopo bisogna premere la spunta per committare

    5. Siamo alla fine. Manca l'ultimo step, forza! 👌

  5. Andare su Commits o Repositories premere la spunta per fare la Push della commit

Ottimo lavoro, sei riuscito a caricare il tuo progetto su GitHub 👏
